免責事項

誤りが多数ある可能性があるためシミュレート結果・スキル内容には特にご注意ください。

当シミュレータの内容は如何なる結果も保証いたしません。
また、利用によって生じた損害に関して、一切責任を負わないことを予めご了承ください。

使い方

マウスカーソルをスキル名に乗せると，スキルの簡易説明が表示されます。

[保存アドレスとスキルリスト]ボタンをクリックするとダイアログが表示されます。

  • ・ 取得スキル一覧とスキルスロットの状態を復元するURLが表示されます
  • ・ 取得スキル一覧をテキストで表示されます

D&Dまたはアイコンをダブルクリックして、スキルスロットにスキルを配置できます。

  • ・ スキルスロットのスキルアイコンをダブルクリックすると解除できます
  • ・ スキルスロット内の位置変更はできないため、解除→再配置をしてください
